# FP-Notla

This project is a note-based e-commerce website, the completion project for a 6-month .NET course I took from Vektörel Bilişim Education Institution.



What does Notla offer students?



\- It allows students to sell their own grades.



\- It allows students to buy grades.



Notla Features:



\- The backend is developed with .NET 8.



\- The frontend is developed with React.



\- MSSQL is used as the database.



\- The number of active users on the site is displayed using SignalR on the front end.
